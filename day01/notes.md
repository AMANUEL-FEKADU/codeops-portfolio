## Terminal commands
**mkdir name** - Make a new directory. 
**touch file** - Create an empty file (or update its timestamp). 
**cat file** - Print a file's contents to the screen. 
**less file** - Page through a long file one screen at a time (q to quit). 
**rm file** - Remove a file. There is no recycle bin.
## Git commands
**git init** - Turn the current folder into a Git repository. 
**git status** - Show what's changed, staged, or ignored — your most-used 
command. 
**git add** - Stage all your current changes for the next commit.
